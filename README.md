# Telco Customer Churn Using SQL & Python

## Project Overview
This project analyzes customer churn for a telecom company using a publicly available Telco Customer Churn dataset.  
The goal is to understand why customers leave and to generate actionable insights for customer retention using SQL for data querying and Python for data analysis and visualization.

---

## Dataset and Project Files
- Dataset : [Dataset.Csv](https://github.com/YasinSyed-2099/Telco-Customer-Churn-/blob/main/Telco-Customer-Churn.csv)
- SQL : [Telco_customer_churn.SQL](https://github.com/YasinSyed-2099/Telco-Customer-Churn-/blob/main/SQL%20Telco_Customer_Churn_Analysis.sql)
- Python : [Telco_customer_churn.ipynb](https://github.com/YasinSyed-2099/Telco-Customer-Churn-/blob/main/Telco%20Customer%20Churn.ipynb)

## Project Process

### **1. SQL Analysis**
- Created a MySQL database **Telco** and table **TelcoCustomerChurn**.  
- Uploaded the CSV dataset into the table.  
- Conducted basic aggregation queries to explore churn patterns.  

---

### **2. Python Analysis (Pandas + Seaborn + Matplotlib)**
- Loaded the dataset using Pandas.
- Performed data exploration.
- Conducted visual analysis.

---


### **3. Conclusion**
This project demonstrates a complete end-to-end data analysis workflow:
- SQL for data storage, cleaning, and aggregation  
- Python for analysis and visualization  
- Actionable insights for business decision-making to reduce churn
